Editable DataTable example - Customize buttons/form using JQuery UI

Preamble

DataTable Editable plugin enables you to apply standard JQuery-ui components in the form. You can define options for each button in the page (add new, delete, confirm add, cancel add) and configure button properties. Also, as form for adding new records is a JQuery-ui dialog you can configure that form too. In the following example is shown how the buttons and form can be customized. Following options can be applied:

  • On the add and delete buttons can be set label and icons (see example on the JQuery UI Buttons). Plugin properties are oAddNewRowButtonOptions and oDeleteRowButtonOptions.
  • On the ok and cancel buttons in the add new record form can be added label and icons, and any other custom properties (e.g. class, name, rel attributes etc).Plugin properties are oAddNewRowOkButtonOptions and oAddNewRowCancelButtonOptions
  • On the add new record popup dialog you can apply all standard dialog properties as shown in the JQuery UI Dialog. Plugin property that will be applied is oAddNewRowFormOptions.

Initialisation code

    $(document).ready(function () {
        var oTable = $('#example').dataTable().makeEditable({
                            sUpdateURL: "UpdateData.php",
                            sAddURL: "AddData.php",
                            sDeleteURL: "DeleteData.php",
                            oAddNewRowButtonOptions: { 
				label: "Add...",
                                icons: { primary: 'ui-icon-plus' }
                            },
                            oDeleteRowButtonOptions: {
				label: "Remove",
                                icons: { primary: 'ui-icon-trash' }
                            },
                            oAddNewRowOkButtonOptions: {
				label: "Confirm",
                                icons: { primary: 'ui-icon-check' },
                                name: "action",
                                value: "add-new"
                            },
                            oAddNewRowCancelButtonOptions: { 
				label: "Close",
                                class: "back-class",
                                name: "action",
                                value: "cancel-add",
                                icons: { primary: 'ui-icon-close' }
                            },
                            oAddNewRowFormOptions: {
				title: 'Add a new browser - form',
                                show: "blind",
                                hide: "explode"
                            }
        });

    });
