Editable DataTable - fixing issue with auto width

Preamble

There is an open issue with using JEditable with DataTable. When you edit cells in the same column, width of the column is increased by 5px. There is still ongoing discussion is the problem in the JEditable or DataTable style. Quick fix is that you set exact width of the editors in the aoColumns array of make Editable call. This code is applied in the example.

Initialization code

To fix an issue with extending width you will need to set widths of each editor as shown below.

$(document).ready( function () {
		$('#example').dataTable().makeEditable({
		                    sUpdateURL: "UpdateData.php",
		                    aoColumns: [
                                       { width: "100px" },
                                       { width: "100px" },
                                       { width: "100px" },
                                       { width: "100px" },
                                       { width: "100px" }
		                      ]
							});
}
